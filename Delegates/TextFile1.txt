делегаты (delegates)

из с++:
указатель на функцию - переменная, содержащая адрес функции (точки входа), 
содержит данные о типах параметров функции

Функтор - обьект (иногда класс), который можно вызвать - применив оператор вызова "()"


ссылатся на внешний модуль
сохранять состояние функции(помнить предыдущий вызов сохраняя состояние обьекта)
испоользовать функции как переменные:
менять их во время выполнения 
создавать из них массивы/коллекции
передавать их как аргументы других функций


Делегаты - элементы языка c# играющие роль указателейна методы

Описание
delegate int Operation(int x);
int - тип возврата
Operation - имя делегата
(int x),тип и кол во параметров

Создание объекта делегата:

Operation operation = Abs;
Operation operation = new Operation(Abs);


Изменение значения 
oper = Sqr
Добавление += Sqr;

выпоолнение последовательно
при выпоолнении нескольких методов возврат делегата = возврат последнего вызоыва
Функции обратного вызова (callback)

реализация событийной модели
	событие -> подписчики(делегаты - укаазатели на методы)
	наступление события -> уведоомление подписчиков (вызов их методов)
